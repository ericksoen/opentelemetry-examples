The resource list below are all locally hosted and intend to help you explore, visualize, and monitor your OpenTelemetry collectors:
  • Jaeger Site to visualize traces
  • Proxies requests to a downstream service
  • Downstream service that receives proxied or direct requests
  • OpenTelemetry Collector Agent Health
  • OpenTelemetry Collector Agent Debug Tool
  • OpenTelemetry Collector Agent Metrics (Prometheus)
  • OpenTelemetry Collector Gateway Health
  • OpenTelemetry Collector Gateway Debug Tool
  • OpenTelemetry Collector Gateway Metrics (Prometheus)
